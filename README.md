
# Plansys2

Fast, Compact, Mindblowing Web-app Builder

## System Requirements

- PHP 7.0
- PHP Extension :
  > PHP mbstring
  > PHP xml
  > PHP zip
  > PHP curl
  > PHP gd
- x64 Server OS
- Nginx Webserver (Recommendation)
- PostgreSQL 10
